Popsy.js

A lightweight JavaScript library for fun web animations. One line, zero setup.

<script src="https://cdn.jsdelivr.net/gh/popsyjs/popsy.js@main/popsy.js"></script>
Components
🎉 Confetti
Burst of colorful dots from the center of the screen. Click a button to preview and see its code.
Popsy.confetti()
Popsy.confetti({ count: 200 })
Popsy.confetti({ colors: ['#f7c948', '#ff6b6b', '#fff'] })
🔔 Toast
Notification popup in the bottom-right corner. Click a button to preview and see its code.
Popsy.toast('Saved successfully!')
Popsy.toast('Something went wrong', { type: 'error' })
Popsy.toast('Did you know?', { type: 'info' })
Popsy.toast('Stays for 6 seconds!', { duration: 6000 })
