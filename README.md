# Popsy.js
A lightweight, fun JS animation library.

## How to use
Add this to your HTML:
  <script src="popsy.js"></script>

## Functions
- Popsy.counter('#el', 1000)             → count up animation
- Popsy.fadeIn('#el')                    → fade in
- Popsy.typewriter('#el', 'Hello!')      → typing effect
- Popsy.shake('#el')                     → shake element
- Popsy.confetti()                       → confetti burst
- Popsy.toast('Message!')               → notification toast
